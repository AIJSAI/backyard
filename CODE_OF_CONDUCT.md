# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) in full.

Short version: be a decent person. Harassment, personal attacks, and bad-faith participation get you removed.

Reports: use GitHub's private "Report abuse" flow on this repository, or contact the maintainer directly (profile links). Enforcement follows the Covenant's enforcement ladder at the maintainer's discretion.
